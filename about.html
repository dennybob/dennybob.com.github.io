Welcome


Thank you for visiting dennybob.com


I created this site to give my machinist colleagues a resource to make calculations that I frequently help them with. The site has also served as a means for me to teach myself web programming. The site is intentionally made to be somewhat lightweight so that pages will still load fast inside metal buildings. Additionally, the calculations are made on the user's device using JavaScript rather than on a server. To use a calculator, type the desired numbers into all input fields and the output will be automatically calculated as you type.


Tips


Here are some tips for using this site:

  • The "DennyBob" logo in the top left corner always takes you to the graphical homepage.
  • The page title will reload the page and thus clear the contents of input boxes.
  • The hamburger menu will drop down links to the other calculators when on a calculator page.
